FUTURE_ML_03 — Customer Support Chatbot
This project is implemented as part of the Future Interns – Machine Learning Internship (Task 3).

The objective of this task is to design and build an AI-powered chatbot that can handle basic customer support queries such as order status, refund policy, and contacting support using Dialogflow.

PROJECT OBJECTIVE
The objective of this project is to:
- Build a rule-based AI chatbot for customer support
- Design conversational flows using intents and training phrases
- Handle common customer queries automatically
- Implement fallback handling for unknown queries
- Test the chatbot using Dialogflow’s built-in interface

TOOLS & TECHNOLOGIES USED
- Dialogflow (No-Code Chatbot Platform)
- Google Cloud Console
- Conversational AI (Intent-based NLP)

DATASET REFERENCE
Customer Support Conversations Dataset (Kaggle)

The dataset was referred to for understanding common customer queries and FAQ-style questions used in chatbot training.

PROJECT WORKFLOW
1. Chatbot Setup
- Created a Dialogflow agent
- Configured language and region settings

2. Intent Creation
- Default Welcome Intent (Greeting)
- Order Status Intent
- Refund Policy Intent
- Contact Support Intent
- Default Fallback Intent

3. Training & Responses
- Added multiple training phrases for each intent
- Configured multiple response variations for natural interaction
- Implemented fallback responses for unmatched queries

4. Chatbot Testing
- Tested chatbot responses using Dialogflow test console
- Verified correct intent matching and responses

KEY FEATURES
- Greeting message for user interaction
- FAQ-based customer support responses
- Smart fallback handling
- Rule-based intent recognition
- Real-time chatbot testing

RESULTS & LEARNINGS
- Gained understanding of chatbot design and conversational flow
- Learned how intent-based NLP works in real-world applications
- Improved skills in designing user-friendly support systems
- Understood the importance of fallback handling in chatbots

HOW TO USE
- Open Dialogflow console
- Test chatbot by typing queries such as:
  - "Where is my order?"
  - "How can I get a refund?"
  - "Contact customer support"

CONCLUSION
This project demonstrates how AI-powered chatbots can automate customer support processes and improve user experience. The chatbot built using Dialogflow successfully handles basic customer queries and simulates a real-world customer support system.
